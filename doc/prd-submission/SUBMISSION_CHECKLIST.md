# Physical Review D Submission Checklist

Use this checklist before and during your submission to [authors.aps.org](https://authors.aps.org).

---

## Before Submission

- [ ] PDF compiles without errors
- [ ] Cover letter PDF ready
- [ ] APS account created at [authors.aps.org](https://authors.aps.org)
- [ ] Title, authors, abstract, PACS numbers prepared
- [ ] Suggested reviewers (optional) — 3–5 names and emails
- [ ] Manuscript not submitted elsewhere

---

## Files to Upload

- [ ] Main manuscript PDF (`main.pdf`)
- [ ] Cover letter PDF (`cover_letter.pdf`)
- [ ] Supplementary material (optional)
- [ ] LaTeX source files (optional)

---

## Submission Form Fields

- [ ] Journal: Physical Review D
- [ ] Article type: Regular Article
- [ ] Title entered
- [ ] All authors with affiliations
- [ ] Corresponding author email and address
- [ ] Abstract copied from manuscript
- [ ] PACS numbers: 12.60.-i, 14.80.Bn, 12.15.Ff, 14.60.Pq, 95.35.+d
- [ ] Keywords (optional)
- [ ] Suggested reviewers (optional)
- [ ] Excluded reviewers (optional)

---

## Final Steps

- [ ] Review submission summary
- [ ] Click Submit
- [ ] Save 7-digit Accession Code for tracking

---

## After Submission

- [ ] Bookmark [Manuscript Status](https://authors.aps.org/Submissions/) page
- [ ] Note submission date
- [ ] Accession Code: _______________
