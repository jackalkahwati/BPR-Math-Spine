Privacy Policy

Last updated: April 2026

Overview

This site (hosted under the Boundary Phase Resonance project by StarDrive Inc.) is a static research website. It does not collect, store, or transmit any personal data. There are no user accounts, cookies, analytics scripts, or tracking pixels.

What we collect

Nothing. This site runs entirely in your browser. No form submissions, no server-side logging, no third-party analytics. The only external network requests are to load fonts (Google Fonts CDN) and the Tailwind CSS CDN for styling.

External services

  • Google Fonts — loads Space Grotesk, Inter, and JetBrains Mono typefaces. Subject to Google's privacy policy.
  • Tailwind CSS CDN — loads the CSS framework. No cookies or tracking.
  • GitHub — links to the public source repository. GitHub's own privacy policy applies if you click through.

PDF downloads

Clicking "Download PDF" or "Open in Browser" for the BPR paper or textbook initiates a direct file download from this server. No download events are logged or tracked.
